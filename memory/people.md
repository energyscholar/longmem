# Key People

> **Optional.** Activate when your project involves multiple people whose roles and context matter. Solo projects may never need this file.

*Not just project collaborators — anyone the AI needs context about: team members, stakeholders, family, mentors, colleagues. Prevents "who's that?" every session.*

---

## Tiers

**Tier 1 — Active:** Daily/weekly interaction. Current collaborators, active stakeholders.
**Tier 2 — Peripheral:** Monthly interaction. Advisors, former collaborators, domain experts.
**Tier 3 — Historical:** Infrequent but contextually important. Mentors, references, key figures in project history.
